# Chronova

Luxury Watch E-Commerce Experience built with React and Tailwind CSS.

## Features

- Responsive Design
- Product Details
- Context API Cart
- Checkout UI
- Framer Motion Animations
- Related Products
- LocalStorage Cart Persistence

## Tech Stack

- React
- React Router
- Tailwind CSS
- Framer Motion
- React Icons

## Installation

npm install

npm run dev
